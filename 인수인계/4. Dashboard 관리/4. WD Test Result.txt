1. 
http://10.177.234.48/w2/bat_summary_bar_chart
BAT Result 차트입니다.
http://10.177.234.48/w2/bat_summary_bar_chart?count=5
이런식으로 count에 값을 주면 해당하는 만큼만 그려집니다.
combination chart를 활용하여 bar차트에 line차트까지 표시하였습니다.
관련 위키 페이지는 http://collab.lge.com/main/display/CCWMP/BAT+Results 입니다.

차트의 오른쪽 하단 Edit를 클릭하면 데이터를 삭제, 입력할 수있는 팝업이 뜨도록 하였습니다. (사용자가 입력할 수 있게)

2. 
http://10.177.234.48/w2/perf_booting_time_line_chart
퍼포먼스 테스트의 부팅타임 차트입니다. 
W2_Dashboard 데이타베이스의 TB_Perf_Result 와 TB_Perf_Image 테이블과 연관이 있습니다.
단, 이부분은 아직 매뉴얼로 데이터를 조작해주어야 합니다 ㅠㅠㅠㅠ
TB_Perf_Image 테이블에 Image 정보를 등록하고 TB_Perf_Result 테이블에 데이터를 입력해야 합니다. 
관련 위키 페이지는 http://collab.lge.com/main/display/CCWMP/Booting_AppLaunch+Time 입니다
위키 페이지보고 데이터를 입력해주면 됩니다.

6.
http://10.177.234.48/w2/system/system_summary_bar.jsp
시스템 테스트 차트 입니다.
이 데이터는 서블릿이 아예 없습니다. (즉, DB 구현이 안되어 있습니다....)
코드에서 데이터를 바꾸어 주셔야 합니다....ㅠ 

xAxis: {
            categories: [
                'Home',
                'Call',
                'Settings',
                'Message',
                'Gallery',
                'Music',
                'IME',
                'Contact',
                'Clock',
                'VoiceMemo',
                'Wi-Fi',
                'Calendar',
                'Bluetooth',
                'Quick Settings'
             ],

X축은 이 코드를 바꾸시면 되고 순서에 맞게

series: [
        // first stack 
		{
		    name: 'Fail',
		    data: [ 25, 26, 43, 39, 8, 31, 0, 50, 8, 12, 19, 5, 12, 0],
		    color: Highcharts.getOptions().colors[2]
		}, {
		    name: 'Pass',
		    data: [ 16, 37, 14, 23, 20, 15, 9, 30, 44, 15, 5, 24, 8, 1],
		    color: Highcharts.getOptions().colors[0]
		}

Fail과 Pass 값을 각각 넣어주시면 됩니다

관련 위키 : http://collab.lge.com/main/display/CCWMP/System+Test+Results
